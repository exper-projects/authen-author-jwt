## Install

```
    pnpm install prisma --save-dev
    pnpm install @prisma/client
```

## Init

```
    npx prisma init
```

## Setup schemas

- Add `DATABASE_URL` variable to `.env.local`
- Create `docker-compose.yaml` at root
- Define table schemas

```
    docket-compose up
    npx prisma migrate dev --name init
```
